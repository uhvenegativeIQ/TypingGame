In order to run this download all the files shown in the main branch and store them to a folder on your desktop, then follow the steps below to run the game.

1) ensure python is installed on your machine first (if not, please download it, the game will not run otherwise) and ensure you have clicked “add to path” when installing, then open the command prompt and type -m pip install -U pygame --user

2) double click the “opening menu” python script in the folder you have created.
